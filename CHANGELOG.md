# Changelog

All notable changes to this project will be documented in this file.

## \[0.1.0\] - 2026-01-05

### Added

- Initial release
- Flex Query API integration for fetching IBKR data
- Automatic exchange rate fetching from multiple APIs
- Excel and JSON report generation
- Support for multiple accounts
- Chinese tax calculation (20% tax rate with foreign tax credit)
- Command-line interface (`ibkr-tax`)
- Comprehensive documentation

### Features

- Parse trades, dividends, and withholding tax data
- Dynamic exchange rates with caching
- Configurable via `.env` file
- Standard Python package structure
